# ☕ Benne n Brew - Local Cafe Menu

A responsive **Local Cafe Menu** built using **HTML, CSS Grid, and JavaScript**. This project provides a clean and user-friendly digital menu that allows customers and café staff to quickly browse menu items.

## 🚀 Features

- Responsive CSS Grid layout
- Search menu items
- Loading indicator
- Empty state handling
- Offline network status detection
- Semantic HTML5
- Accessible form elements
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- JavaScript (ES6)

## 📱 Responsive Design

- 💻 Desktop (3 columns)
- 📱 Tablet (2 columns)
- 📱 Mobile (1 column)

## 🎯 Project Objective

To create a responsive digital café menu that enables customers and café staff to browse menu items quickly while providing search functionality, loading states, and offline network detection.
